## 📖 什么是「酒馆」

### SillyTavern 简介

[SillyTavern](https://sillytavern.app/) 是目前最流行的开源 AI 角色扮演前端之一。它让用户可以导入「角色卡」，让 AI 扮演各种角色——小说人物、原创角色、历史人物，随你喜欢。

围绕它形成了一个庞大的社区生态，有海量角色卡、预设模板、世界观设定在流通。

**重要的是：你不需要安装 SillyTavern 本体。** 这套生态可以直接在 Discord 里使用，所有资源无缝兼容。如果你之前是酒馆用户，已有的角色卡和预设可以直接迁移过来。

### 去哪里获取角色卡和资源

#### 中文用户推荐优先去这里：

- **类脑** — 国内最活跃的酒馆社群，大量中文角色卡和预设
- **旅程** — 另一个优质中文社群，资源丰富，有教程和交流

#### 国际站点：

- [Chub.ai](https://chub.ai/) — 最大的角色卡分享社区之一
- [CharacterHub](https://www.characterhub.org/) — 另一个角色卡社区
- [SillyTavern 文档](https://docs.sillytavern.app/) — 如果你想深入了解这套体系

> 嘛，中文角色卡的质量参差不齐，但类脑和旅程里优质内容不少。去翻翻吧。

## 🧩 三个核心概念

在开始之前，先搞清楚这三样东西是什么——这会让你少走很多弯路。

### 🃏 角色卡（Character Card）

角色卡定义了 AI 要扮演的角色。它包含：

- **角色名称和描述** — 这个角色是谁
- **性格设定** — 说话方式、喜好、行为逻辑
- **开场白（可能有多条）** — 每条对应一个不同的故事起点
- **对话示例** — 帮助 AI 理解角色的语气和风格
- **创作者备注** — 给 AI 的额外指引

格式通常是 **PNG 图片**（角色立绘里嵌入了设定数据）或 **JSON 文件**。支持 V2 和 V3 两个版本。

> 一张好的角色卡，会让 AI 非常"入戏"。一张写得烂的……嘛，换一张就好。

### 📋 预设（Preset）

预设控制的是 AI 的"演技"——它定义了 AI 该如何理解和执行角色卡里的设定，以及提示词的组装方式和风格倾向。

简单理解：**角色卡决定"角色是谁"，预设决定"AI 怎么演"。**

同一张角色卡配不同的预设，体验会截然不同。有些预设擅长文学向 RP，有些针对特定模型做了优化。

> 预设不是必须的——不配置也能玩，只是体验会参差不齐。有好的预设时，差距是很明显的。这是客观事实，不是在夸什么。

### 🌍 世界书（World Book / World Info）

世界书是背景设定集。它包含大量词条，每个词条有：
- **关键词** — 触发条件
- **内容** — 相关的背景信息

当对话中出现匹配的关键词时，对应的背景信息会**自动注入** AI 的上下文。

比如世界书里有一条关于"魔法议会"的词条，当对话中提到这个词，AI 就会自动获得相关设定，不需要你每次手动解释。

> 世界书让 Roleplay 的世界观更丰富、更一致。角色多、剧情复杂的时候尤其有用。

## 🚀 开始 Roleplay：完整流程

### 第一步：创建 Roleplay 线程

RP 在独立的**线程**中进行，和普通聊天隔离。这是一切的起点。

**公开线程**：
```
/roleplay new title:线程标题（可选）
```

任何人都可以看到和参与。

**私密线程**：
```
/roleplay new-private title:线程标题（可选）
```

只有你（和被邀请的人）能看到。

### 第二步：加载角色卡（唯一必须的操作）

```
/character use character:角色名
```

输入时会有自动补全提示，从你的角色库中选择。

> **这是唯一必须做的配置。** 模型、预设等都有默认值，不设置也能直接玩。

如果你还没有角色卡，需要先上传——参见下方「角色卡管理」章节。

### 第三步：选择开场白

加载角色卡后，系统会展示角色卡里包含的开场白。

许多角色卡作者会写**多条开场白**，每条对应不同的故事起点或场景。系统提供「上一条」/「下一条」按钮，翻阅选择一个你喜欢的。

选定后，RP 正式开始。

### 第四步：开始对话

直接在线程里发消息，Bot 会以角色身份回复。就像和真人文字 RP——只不过对面是 AI。

## 🔁 对话中的重要操作

### Retry：重新生成回复

**这是 Roleplay 里最常用的操作之一。**

AI 的回复可能不如预期——人设偏了、内容不对、或者直接拒绝生成。这时：

1. **编辑你发出的最后一条消息**（在 Discord 里直接编辑）
2. **使用 `/regen` 命令**（或前缀 `!retry`）重新生成 AI 的最后一轮回复

Bot 会基于你修改后的内容重新生成，相当于"这一轮重来"。

> ⚠️ Retry 只对**最后一轮对话**有效，无法回溯更早的内容。

这个命令在实际 RP 里非常实用，善用它可以大幅提升质量。

### 对话总结与 Retry Summary

在长 RP 中，系统会自动生成结构化的剧情总结（角色状态、情节进展等），确保 AI 始终记得之前发生了什么。

如果某轮自动总结出了偏差，可以点击消息上的「重新总结」按钮重新生成，或使用：
```
/retry-summary
```

日常使用中不常需要这个，但出问题时它很有用。

## 使用 Bot 创作角色卡

Bot 内置了**对话式角色卡创作工坊**，无需借助 SillyTavern，直接在 Discord 里就能创作符合标准 V3 格式的角色卡。

### 如何进入创作模式

在 #角色卡创作助手 频道点击**「创建或修改角色卡」**按钮面板。点击按钮后，Bot 会为你自动创建一个**私有线程**（仅你和 Bot 可见），线程中提供三个选项：

- **创建新角色卡** — 从零开始，AI 引导你逐步填写所有字段
- **加载已有角色卡** — 从你的角色库中选择一张，继续修改
- **上传角色卡文件** — 发送 `.png` 或 `.json` 文件，Bot 自动解析后进入编辑模式

如果你已在创作线程中，也可以直接使用 `/character-create start` 重新显示选项菜单。

### 创作过程

进入创作或编辑状态后，直接在线程中用自然语言描述你的想法，Bot 会帮你填写和完善角色卡各个字段。例如：

- 描述角色的外貌、性格、背景故事
- 修改某个字段，例如「把开场白改成更神秘的风格」
- 请 AI 补全缺失的内容，例如「帮我想几条对话示例」

每轮对话后，Bot 会显示**本轮更新了哪些字段**以及**当前整体完成进度**。角色卡数据在每轮对话后**自动保存**到你的账户，关闭线程也不会丢失进度，下次打开线程或加载同一张卡可以继续编辑。

### 预览与导出

随时使用关键词“预览”、“导出”触发当前正在编辑的角色卡的状态预览和导出功能。

或可以在创作线程中使用以下命令：

- `/character-create preview` — 查看当前角色卡所有字段的完整内容
- `/character-create export` — 下载 JSON 格式的角色卡文件，可直接导入 SillyTavern 使用
- `/character-create cancel` — 解除当前线程与角色卡的绑定（不会删除已保存的角色卡）

创作完成后，角色卡会保存到你的角色库，直接用 `/character use` 加载即可开始 Roleplay。

## ⚙️ 可选配置（提升体验）

以下不是必须的，但配置之后体验会明显更好：

### 切换 AI 模型

不同模型在 RP 中表现差异很大。

- **去不同频道开 RP** — 酒馆区每个频道绑定了不同的模型（Gemini、Claude、ChatGPT、DeepSeek 等）
- **手动切换** — 使用 `/chat-setting` 命令

同一张角色卡在不同模型下会有截然不同的风格，多试试找到你喜欢的。

> 哼……我不会推荐哪个模型最好。因为这取决于你想要什么样的体验。自己试才有说服力。

### 上传并使用预设

从类脑、旅程等社群获取针对特定模型优化的预设文件，然后：

```
/preset upload file:（拖入预设文件）
/preset use
```

通过交互式菜单选择要应用的预设。

### 添加世界书

为 RP 增加更丰富的世界观背景：

```
/world upload file:（拖入世界书文件）name:名称（可选）
```

上传后自动工作——对话中触发关键词时自动注入背景信息，不需要手动操作。

### 设置主控角色（你扮演的角色）

```
/character add-user user:角色名称 personality:性格描述（可选）
```

设定后，AI 会将你视为这个角色，而不是以"用户"身份互动。

切换已创建的主控角色：
```
/character choose-user
```

## 💡 RP 线程的特性

了解这些特性能帮你更好地使用 RP 线程：

| **特性** | **说明** |
|-----|-----|
| 独立上下文 | 每个线程有自己独立的对话历史，不受其他频道影响 |
| 自动继承配置 | 线程继承父频道的模型、预设、角色卡等设置 |
| 自动剧情总结 | 长对话中自动生成结构化总结，保持连贯性 |
| 好感度隔离 | RP 线程中好感度系统自动禁用，角色扮演不会影响你和 Bot 本体的关系 |

## 💰 积分消耗

Roleplay 中的 AI 对话与普通对话相同，按模型档位消耗积分：

| **档位** | **积分**/次 | **代表模型** |
|-----|--------|---------|
| ⚡ 轻量 | 1 | Gemini Flash、GPT-5 Mini、DeepSeek V3 |
| 💙 中档 | 2 | Claude Haiku、DeepSeek R1、GLM-5、Gemini Pro |
| 💜 专业 | 4 | GPT-5/5.1、Grok-4 |
| 🧡 旗舰 | 8 | Claude Sonnet、GPT-5.4 |
| 💎 顶级 | 12 | Claude Opus |

> 默认模型每条消息只需 1 积分。日常签到 + 聊天就能覆盖日常 RP 消耗。

## 🎯 推荐流程

### 初次体验（最快路径）

1. 去类脑、旅程社群或 [Chub.ai](https://chub.ai/) 下载一张感兴趣的角色卡 PNG
2. 使用 `/character upload` 上传
3. 使用 `/roleplay new-private` 创建私密线程
4. 使用 `/character use` 加载角色卡
5. 选择喜欢的开场白，开始对话

**就这五步，没有更简单的了。**

### 进阶玩法

- **多模型对比** — 同一张角色卡在不同模型下的表现差异很值得探索
- **预设调优** — 找一个与你使用的模型适配的预设，体验提升明显
- **世界书加持** — 剧情复杂、角色多的 RP，世界书是必备的
- **主控角色** — 设定自己的角色身份，让故事更有代入感
- **私密线程** — 只给自己看的故事，用 `/roleplay new-private`

## ❓ 常见问题

### **Q：我没用过 SillyTavern，能直接玩吗？**
A：完全可以。不需要安装任何东西，去社群下载一张角色卡，上传就能开始。

### **Q：角色卡、预设、世界书去哪里找？**
A：中文用户推荐先去**类脑**和**旅程**社群，国际站推荐 [Chub.ai](https://chub.ai/)。

### **Q：预设和世界书是必须的吗？**
A：不是。角色卡是唯一必须的，其余都是可选的增强配置。但用了之后体验差距很明显。

### **Q：AI 回复不好怎么办？**
A：用 `/regen`（或 `!retry`）重新生成。想要更好的结果，先编辑你的最后一条消息再 Retry。

### **Q：不同频道的角色卡配置会互相影响吗？**
A：不会。每个频道/线程的配置完全独立。

### **Q：RP 中的对话会影响好感度吗？**
A：不会。RP 线程中好感度系统自动禁用，角色扮演不会影响你和 Bot 本体的关系。

### **Q：我可以自己创建角色卡吗？**
A：可以。可以用 SillyTavern 创建并导出，或者使用各种在线角色卡编辑器。

### **Q：想和其他人一起 RP 怎么办？**
A：使用 `/roleplay new` 创建公开线程，或者用 `/character add-user` 邀请其他用户加入 RP 并设定他们的角色身份。
