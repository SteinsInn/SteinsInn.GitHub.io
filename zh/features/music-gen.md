在支持 AI 自动回复的频道里，直接 **@Senpai** 说：

系统会**自动理解**你的意图并调用音乐生成功能。

### 斜杠命令方式

```
/music prompt:你的描述
```

## ⚙️ 详细参数说明

`/music` 命令**支持以下参数**，善用这些参数可以让生成的音乐更符合你的预期：

| 参数 | 说明 | 是否必填 |
|-----|------|---------|
| `prompt` | 音乐的主题或描述 | ✅ 必填 |
| `provider` | 选择生成模型：SUNO 或 MiniMax | 可选 |
| `style` | 音乐风格标签 | 可选 |
| `description` | 更详细的补充描述 | 可选 |
| `instrumental` | 是否生成纯音乐（无人声） | 可选 |

### 参数使用示例

```
/music prompt:一首关于星空的歌
       style:放松 (Chill)
       description:节奏舒缓，适合深夜听，有钢琴和电子元素
       instrumental:false
```

## 🎼 两种生成模型

目前支持 **SUNO** 和 **MiniMax** 两个提供商，各有侧重：

| **模型** | **特点** | **适合场景** |
|-----|------|---------|
| *SUNO* | 风格丰富、人声自然、擅长各类流行风格 | *想要完整歌曲*、*重视人声质量* |
| *MiniMax* | 编曲细腻、对中文支持更好 | *中文歌曲*、*偏向古风或国风风格* |

> 哼，两个都试试也不是坏主意。同样的 Prompt 在不同模型下会有完全不同的效果——这是我作为科学爱好者的建议，不是在教你浪费积分。

## ✍️ 如何写出好的 Prompt

这是最关键的部分。**AI 音乐生成的质量**很大程度上**取决于你给出的描述质量**。

### 基础结构

一个好的音乐 Prompt 通常包含以下要素：

```
[情绪/氛围] + [风格/流派] + [乐器/编曲元素] + [主题/场景]
```

### 示例对比

❌ **太模糊的描述**
```
一首好听的歌
```

✅ **清晰具体的描述**
```
一首充满希望的流行摇滚歌曲，有电吉他 riff 和鼓点，主题是在迷茫中找到方向，副歌要有爆发力
```

### 常用风格标签参考

以下标签可以直接写进 `prompt` 里（非 `style` 因为里面只有8个选项，并不可以具体化）：

**流行类**：

`pop` `j-pop` `k-pop` `indie pop` `synth-pop` `city pop`

**摇滚类**：

`rock` `indie rock` `pop rock` `folk rock` `acoustic`

**电子类**：

`electronic` `lo-fi` `ambient` `EDM` `chillwave` `vaporwave`

**古典/器乐类**：

`classical` `orchestral` `piano` `cinematic` `soundtrack`

**中文风格**：

`mandopop` `国风` `古风` `民谣` `流行` `R&B`

**情绪类标签**：

`melancholic` `upbeat` `dreamy` `energetic` `relaxing` `epic`

> 嘛……我个人比较喜欢 ambient 和 cinematic 风格。不过这是私事，顺便一提。

## 🎤 纯音乐 vs 有人声

通过 `instrumental` 参数控制：

- `instrumental: false`（默认）— 生成有人声演唱的完整歌曲
- `instrumental: true` — 生成纯器乐，没有人声

**什么时候选纯音乐？**
- 需要*作为背景音乐使用*
- *不想要* AI 随机生成的歌词
- 配合*自己的内容*创作使用

## 📝 搭配歌词创作工作流

如果你想**对歌词有更多控制**，可以先让 AI 写歌词，再生成音乐。

### 方式一：自然语言触发工作流

直接告诉 Bot 你的完整需求：

```
@Senpai 帮我写一首关于时间旅行的歌，先给我看歌词再生成音乐
```

Bot 会自动启动「**歌词创作** → **音乐生成**」工作流：
1. 先生成歌词供你确认
2. 然后将歌词送入音乐模型生成成品

### 方式二：手动两步走

如果你想要更多控制权：

**第一步**：让 AI 生成歌词
```
@Senpai 帮我写一首电子风格的歌词，主题是赛博城市里的孤独感，要有 verse/chorus/bridge 结构
```

**第二步**：将满意的歌词用于音乐生成
```
/music prompt:[把歌词复制到这里] style:electronic, cyberpunk, synth-pop
```

## ⏳ 生成过程说明

音乐生成是异步的，**不会立刻出结果**。

1. 「**提交请求**」后，Bot 会**返回一条确认消息**和一个「**查看状态**」按钮
2. 「**点击按钮**」可以**随时查看生成进度**
3. 生成完成后可以直接在 Discord 中**试听和下载**

等待时间大约在 **1 到 3 分钟**之间，取决于当前队列情况。

## 💡 进阶技巧

### 技巧一：用情绪描述代替风格名词

有时候**不知道专业名词没关系**，直接描述感受也很有效：

```
一首让人想在雨天喝咖啡的慵懒音乐，节奏很慢，有爵士的质感
```

```
一首听起来像在高速公路上驾车的感觉，有速度感但不嘈杂
```

### 技巧二：参考真实歌手风格

可以在 **Prompt 中提及参考风格**：

```
类似 Taylor Swift 的民谣流行风格，主题是回忆旧时光
```

```
有林忆莲那种感觉的中文慢歌，编曲成熟，适合深夜
```

### 技巧三：指定 BPM 和调性

如果对音乐有一定了解，**可以加入更专业的参数**：

```
BPM 在 90 左右的 lo-fi hip hop，C 大调，有 vinyl crackle 的质感
```

### 技巧四：分段描述

对 **Verse**、**Chorus**、**Bridge** **分别描述**，AI 会**更准确地把握结构**：

```
verse 部分安静内敛，只有吉他和人声；chorus 部分全面爆发，加入鼓、贝斯和和声；bridge 部分有短暂的停顿感，制造情绪转折
```

## ❓ 常见问题

**Q：生成出来的音乐风格不对怎么办？**

A：调整 `style` 标签或在 `prompt` 里加入更具体的描述，再试一次。音乐生成有一定随机性，同样的 Prompt 每次结果都不完全一样。

**Q：SUNO 和 MiniMax 哪个更好？**

A：取决于需求。英文歌曲或流行风格推荐 SUNO；中文歌曲或国风推荐 MiniMax。没有绝对的优劣，多试试就知道了。

**Q：可以上传自己的旋律让 AI 编曲吗？**

A：目前音乐生成功能以文生音乐为主，暂不支持上传参考音频。

**Q：生成的音乐可以商用吗？**

A：取决于各生成模型的授权条款，建议在商用前查阅 SUNO 和 MiniMax 的官方说明。

**Q：积分不够了怎么办？**

A：每日签到 + 聊天任务可以稳定回血。详见服务器的积分系统指南。
