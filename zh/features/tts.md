## ⚙️ 参数说明

`/voice` 命令支持以下参数：

| **参数** | **说明** | **是否必填** |
|-----|------|---------|
| `text` | 要转换为语音的文字内容 | ✅ 必填 |
| `voice` | 选择音色 | 可选（有默认值） |
| `format` | 输出音频格式 | 可选 |
| `emotion` | 情感语音（实验性功能） | 可选 |


## 🎤 音色选择

语音合成最多支持 **25 种预设音色**，涵盖不同风格和语言。

### 如何选择音色

在 `/voice` 命令中使用 `voice` 参数指定：

```
/voice text:你好，欢迎光临 voice:音色名称
```

输入 `voice` 参数时会有自动补全提示，直接从列表中选择你想要的音色。

> 嘛，不知道选哪个的话，就先用默认的试试。不满意再换——反正换音色又不收额外费用。

### 选音色的参考方向

不同音色适合不同的使用场景，以下是一些参考方向：

| **场景** | **推荐方向** |
|-----|---------|
| 日常对话 / 陪伴 | 自然、亲切的女声或男声 |
| 正式播报 / 朗读 | 清晰、沉稳的播音风格 |
| 角色扮演配音 | 根据角色气质选择对应音色 |
| 故事朗读 | 有感情起伏的叙述风格 |
| 纯粹趣味 | 任何你觉得好玩的音色 |

## 📁 输出格式

通过 `format` 参数可以指定音频文件的格式：

| 格式 | 特点 | 适合场景 |
|-----|------|---------|
| `mp3` | 体积小，兼容性最好 | 日常使用、分享 |
| `wav` | 无损音质，体积较大 | 需要高质量音频时 |
| `ogg` | 开源格式，体积小 | 特定播放器或平台 |
| `flac` | 无损压缩 | 对音质要求极高的场景 |

不指定时默认输出 `mp3`。


## 💡 写出好文本的技巧

语音合成的效果很大程度上取决于你输入的文本质量。

### 基本原则

- **标点符号很重要** — 逗号、句号会影响语气停顿，该加的地方加上
- **避免符号堆砌** — 表情符号、特殊字符可能会被读出来或者读错
- **数字的处理** — 数字建议写成汉字或带单位（"三百元"而非"300"），避免读音歧义
- **英文混排** — 中英文混合的文本大部分情况下都能正常处理，但全英文长句建议选择英文音色

### 示例对比

❌ **可能导致问题的文本**
```
今天天气棒棒哒☀️！气温28℃，风速3m/s，记得SPF50+防晒！
```

✅ **更适合 TTS 的文本**
```
今天天气很好，气温二十八度，风速较小，记得做好防晒。
```

### 分段朗读技巧

如果文本较长，适当添加标点可以让节奏更自然：

```
/voice text:欢迎来到次元旅社。这里是连接各个世界线的特殊空间，每一位旅者都能在这里找到属于自己的故事。
```

## 🎭 情感语音（实验性）

`emotion` 参数是实验性功能，可以尝试为语音赋予特定的情感色彩。

```
/voice text:你回来了啊…… emotion:True
```

::: warning
> ⚠️ 标注为实验性，效果因音色和文本内容而异，不保证每次都达到预期。多试几次，效果满意了再用。

注意，本功能仅有 "True" 和 "False"

:::

## 🔗 搭配其他功能使用

TTS 单独用当然没问题，但和其他功能组合起来更有意思：

### 搭配 AI 对话

让 AI 生成一段文字，再转换为语音：

```
@Senpai 用语音给我讲一个短故事
```

Bot 会自动走「AI 生成文字 → 语音合成」的工作流，不用分两步操作。

### 搭配角色扮演（voice-roleplay 工作流）

在已加载角色卡的频道中，可以让角色"开口说话"：

```
@Senpai 用语音回复我
```

这会触发专门的**角色语音对话工作流**：
1. AI 以角色身份生成对话文本
2. 自动清洗格式标记（去掉动作描写的星号等）
3. 转换为语音输出

让角色的声音真正"活"起来——唔，这个功能嘛……承认一下，确实挺有意思的。

### 搭配音乐生成

先写歌词、生成音乐，再用 TTS 朗读歌词预览：

```
/voice text:[把生成好的歌词复制进来]
```

## ⏱️ 生成速度

与视频生成不同，TTS 通常是**近实时**的。

- 短文本（几十字）：通常 **1–3 秒**内返回
- 长文本（几百字）：可能需要 **5–15 秒**
- 极长文本：视内容长度而定

生成完成后，音频文件会直接在 Discord 聊天中显示，可以直接播放或下载。

> 比视频那种等几分钟的体验好多了，嘛……虽然视频生成也挺值的就是了。

## ❓ 常见问题

### **Q：某些字读错了怎么办？**
A：把容易读错的词替换成同音的正确写法，或者改用汉字拼写代替。这是目前 TTS 技术的共性问题，没有完美方案，只能通过调整文本来规避。

### **Q：可以指定语速或音调吗？**
A：目前主要通过 `voice`（音色）和 `emotion`（情感，实验性）来调整风格，不支持直接设置语速/音调参数。部分音色天生语速较快或较慢，可以通过换音色来找到合适的节奏。

### **Q：支持方言或小语种吗？**
A：取决于选择的音色，部分音色支持粤语、台湾腔等变体，以及日语、英语等外语。具体请在音色列表中查看各音色的支持范围。

### **Q：生成的音频可以下载吗？**
A：可以。在 Discord 中音频文件会作为附件发送，直接点击下载即可。

### **Q：文本长度有限制吗？**
A：有上限，通常建议单次不超过 **500 字**以内，过长的文本建议分段处理。

### **Q：Roleplay 中的角色语音是自动的吗？**
A：不是完全自动的，需要你明确触发（如说"用语音回复我"）。正常 RP 对话默认是文字输出，你随时可以选择是否要语音。
